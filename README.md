# 介绍

`牧羊人`是一款完全公益性质的综合型QQ群机器人。

`牧羊人`所有服务完全免费，并不存在商业收费的现象，您可以将牧羊人拉入自己的QQ频道享受其提供的服务，但请注意您在使用的过程中需要遵循其[使用条款](docs/快速开始/04.使用须知.md)。
